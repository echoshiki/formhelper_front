# 自定义表单平台

[后端仓库](https://github.com/echoshiki/formhelper) ｜ [前端仓库](https://github.com/echoshiki/formhelper_front)

## 介绍

- 一个自定义表单平台的项目
- 前端 React + Vite + TailwindCSS + Shadcn
- 后端 Webman + DNMP
- 可以创建多种自定义字段的表单
- 分享用户填报
- 个人用户管理面板
- 充值并购买创建额度
- 适配移动端
